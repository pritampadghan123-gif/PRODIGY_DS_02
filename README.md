# PRODIGY_DS_02 - Titanic EDA

## Objective
Perform Exploratory Data Analysis (EDA) on the Titanic dataset to understand survival patterns.

## Dataset
- Titanic Dataset
- Source: https://www.kaggle.com/c/titanic/data

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Steps Performed
- Loaded dataset
- Checked missing values
- Cleaned data
- Performed Exploratory Data Analysis
- Created visualizations
- Found key insights

## Key Insights
- Most passengers belonged to 3rd class.
- Female passengers had a higher survival rate.
- Passenger class and fare influenced survival.
